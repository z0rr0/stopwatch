# stopwatch

Simple console stopwatch tool.

Example:

```
S - start/stop, L - lap,  Q - quit
--
   0:00:10.9870455630
   0:00:15.15121302855
   0:00:19.18943639537
--
   0:00:02.2431599589
--
   0:00:05.5445222805
   0:00:06.5832889410
```
